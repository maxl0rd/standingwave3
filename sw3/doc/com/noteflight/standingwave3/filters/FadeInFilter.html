Classpublic class FadeInFilter
InheritanceFadeInFilter AbstractFilter Object

FadeInFilter passes the signal unchanged after fading it in. Many times a fade can be used in place of an Envelope + AmpFilter and will be much more efficient, since it only calculates during the fading part.
