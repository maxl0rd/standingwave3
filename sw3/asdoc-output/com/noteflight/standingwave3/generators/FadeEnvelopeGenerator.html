FadeEnvelopeGenerator()Constructor
public function FadeEnvelopeGenerator(descriptor:AudioDescriptor, t:Number, type:String)

A FadeInEnvelopeGenerator generates a sample table that describes the envelope shape for a fade in, and can be passed to a FadeInFilter. One generator can be reused across any voices that have the same envelope shape, and does not need to be cloned.
